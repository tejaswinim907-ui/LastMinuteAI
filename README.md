# 🚀 LastMinute AI

LastMinute AI is an AI-powered study assistant designed to help students prepare efficiently for exams. It combines intelligent learning tools with task management to make studying faster, smarter, and more organized.

## ✨ Features

- 📚 AI-powered Notes Summarization
- ❓ AI Quiz Generation
- ✅ Task Management
- 📅 Deadline Tracking
- 🤖 AI Study Assistant
- 📱 Responsive User Interface

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/tejaswinim907-ui/LastMinuteAI.git
```

### Navigate to the project

```bash
cd LastMinuteAI
```

### Install frontend dependencies

```bash
cd frontend
npm install
npm run dev
```

### Run backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```
